Passo 1: Preparar a sua pastaVá para a sua Área de Trabalho.Clique com o botão direito, escolha Novo -> Pasta e dê o nome de MeuPrograma.Abra essa pasta. Deixe ela aberta aí na sua tela.Passo 2: O comando mágico que junta tudoO próprio sistema do MSYS2 tem uma ferramenta que descobre e copia todas as DLLs para você sem que você precise procurar nada.Abra o seu terminal MSYS2 UCRT64.Cole o comando abaixo para entrar na pasta que você acabou de criar na Área de Trabalho e aperte Enter:bashcd ~/Desktop/MeuPrograma
Use o código com cuidado.Agora, cole este comando para compilar o seu código direto para dentro dessa pasta (substitua teste.c pelo caminho real do seu arquivo se ele estiver em outro lugar):bashgcc ~/Desktop/teste.c -o programa.exe `pkg-config --cflags --libs gtk4`
Use o código com cuidado.Aqui está o segredo: Cole este comando exatamente como está e dê Enter. Ele vai varrer o sistema e copiar todas as DLLs necessárias para dentro da sua pasta automaticamente:bashcp /ucrt64/bin/*.dll . 2>/dev/null || true
Use o código com cuidado.Olhe para a pasta MeuPrograma na sua Área de Trabalho. Ela agora está cheia de arquivos .dll e tem um arquivo chamado programa.exe lá dentro. Se você der dois cliques no programa.exe, ele vai abrir a janela perfeitamente!
